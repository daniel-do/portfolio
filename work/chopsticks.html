Chopsticks!

Chopsticks! is a sandbox interactive environment where the user can pick up various objects using chopsticks and either throw or drop these objects. Anything from a small box to a massive building cane be picked up and thrown!

I developed and designed the dart-like object throwing mechanic and the target arena.
